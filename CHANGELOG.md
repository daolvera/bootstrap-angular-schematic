# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-11

### Added
- Initial release of Bootstrap Angular Schematic
- Angular 19+ support with standalone components
- Bootstrap 5+ integration with ng-bootstrap
- Pre-built services:
  - SpinnerService for loading states
  - ModalService for dialogs
  - NavigationService for sidebar control
  - NotificationService for toast messages
- Pre-built components:
  - HeaderComponent with responsive navigation
  - FooterComponent with multi-column layout
  - SidebarComponent for mobile navigation
  - LayoutComponent for application structure
  - SpinnerComponent for loading indicators
  - NotificationsComponent for toast displays
  - SampleModalComponent for demonstration
- Sample pages:
  - HomeComponent with service demos
  - AboutComponent with feature list
  - NotFoundComponent for 404 errors
- Custom SCSS structure with Bootstrap variable overrides
- Responsive design with mobile-first approach
- Production-ready configuration
- Comprehensive documentation

### Features
- ✅ Standalone components (Angular 19+)
- ✅ Bootstrap 5+ fully integrated
